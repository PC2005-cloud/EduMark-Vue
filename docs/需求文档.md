# 基于大模型的中小学作业批改系统 — 需求文档

## 1. 项目概述

### 1.1 项目背景
开发一套能够自动识别和批改中小学生作业的智能批改系统，利用计算机视觉、OCR 技术和大语言模型，实现从作业图像采集到最终评分批注的全自动化流程。

### 1.2 项目目标
- 支持作业图片上传和 MinIO 对象存储
- 双识别引擎：阿里云 OCR（切题 + 题目识别）或百炼视觉模型（端到端图像理解）
- 调用大模型 API 对学生答案进行智能批改，输出得分、评语和解题分析
- 基于向量知识库的检索增强批改（RAG）
- 支持教师上传知识文档（PDF/PPT/Word），自动解析、向量化后存入 Qdrant
- 完整的用户管理体系：多角色（管理员/教师/学生）注册、认证和权限控制
- Celery 异步任务处理耗时操作（批改作业、文档解析）

### 1.3 技术栈
| 层面 | 技术选型 |
|------|----------|
| 后端框架 | FastAPI (Python) |
| 图像增强 | Pillow（对比度 + 锐度增强） |
| 识别引擎一（传统 OCR） | 阿里云 RecognizeEduPaperCut + RecognizeEduQuestionOcr |
| 识别引擎二（视觉理解） | 百炼平台视觉理解模型（Qwen-VL 系列） |
| 批改引擎 | 百炼平台大模型（Qwen-Plus） |
| 向量数据库 | Qdrant |
| 向量化模型 | text-embedding-v4（1536 维） |
| 文档解析 | MinerU（PDF/Ppt/Word 解析为结构化文本） |
| 用户认证 | JWT + bcrypt 密码哈希 |
| 关系数据库 | MySQL + SQLAlchemy |
| 任务队列 | Celery + Redis |
| 对象存储 | MinIO |
| 部署 | Docker Compose |

| 功能 | 描述 |

| 查看配置 | 查看当前用户配置 |

### 2.5 图像采集模块（前端）

| 功能 | 描述 |
|------|------|
| 摄像头拍摄 | 支持调用摄像头拍摄作业图像（前端实现） |
| 图片上传 | 支持上传已有图片文件（jpg / png） |
| 批量导入 | 支持一次性导入多张作业图片 |
| MinIO 存储 | 上传图片自动存入 MinIO |

### 2.6 图像预处理

提交作业时系统自动对图片进行轻度增强，以提升 OCR 识别率。

| 功能 | 方法 |
|------|------|
| 对比度增强 | ImageEnhance.Contrast (1.2x) |
| 锐度增强 | ImageEnhance.Sharpness (1.1x) |
| 输出格式 | PNG |

用户可在配置中关闭图像增强。

### 2.7 作业批改模块

#### 2.7.1 提交作业
1. 用户上传作业图片（支持多张）
2. 图片存入 MinIO，创建 Task 记录
3. 投递 Celery 异步任务，立即返回 task_id
4. 前端轮询 `/homework/status/{task_id}` 获取状态

#### 2.7.2 批改流程
```
Task(status=pending)
  → 读取用户配置（识别模式/模型/开关）
  → 从 MinIO 下载图片
  → 图像预处理（可选，对比度+锐度增强）
  → 多线程 OCR 识别
    阿里云 OCR：切题 → 题目识别
    或百炼视觉：多图分页识别
  → 保存题目到数据库
  → 多线程批改
    知识检索（可选）：向量化 → Qdrant 搜索 → Top-K 知识片段
    LLM 批改：题目 + 学生答案 + 知识片段 → 得分/评语/分析
  → 保存批改结果 + 知识引用记录
  → Task(status=completed)
```

#### 2.7.3 查询结果

| 功能 | 描述 |
|------|------|
| 查看状态 | 查询 task 当前状态 |
| 查看结果 | 查询完整批改结果（题目、答案、得分、评语、位置、知识引用） |
| 任务列表 | 分页查询所有任务 |

### 2.8 知识库模块（RAG 检索增强）

#### 2.8.1 上传知识文档

教师上传 PPT/PDF/Word 文档，系统自动完成解析和向量化。

```
上传文档 → 存 MinIO → 创建 Knowledge 记录
  → Celery 异步任务：
    从 MinIO 下载 → MinerU 解析 → 轮询结果
    → 提取文本 → 分块（2000 字/块）
    → 图片转文字描述（百炼视觉模型）
    → Embedding 向量化（text-embedding-v4）
    → 写入 Qdrant 向量库
    → 更新 Knowledge(status=completed)
```

#### 2.8.2 知识文档管理

| 功能 | 描述 | 权限 |
|------|------|------|
| 文档列表 | 分页查询知识文档 | 登录 |
| 文档详情 | 查看文档信息 | 登录 |
| 上传文档 | 上传并异步解析 | 教师/管理员 |
| 删除文档 | 删除记录 + MinIO 文件 + Qdrant 向量 | 教师/管理员 |

#### 2.8.3 检索增强批改流程

```
题目文本
  → Embedding 向量化
  → Qdrant 搜索 Top-K 相似知识块（支持按学科过滤）
  → 知识片段注入大模型 Prompt
  → LLM 批改 → 评分 + 评语 + 知识引用
```
